# Role: dev-environment
Configures a professional development environment on Ubuntu and Alpine hosts.

## Tasks
- **Shell**: Installs and configures Zsh as the default shell.
- **Python**: Adds the `deadsnakes` PPA on Ubuntu to provide the latest Python versions.
- **Utilities**: Installs a curated list of development tools (`fzf`, `ripgrep`, `build-essential`, etc).
