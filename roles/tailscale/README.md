# Role: tailscale
Installs and configures Tailscale for secure networking.

## Tasks
- **Installation**: Handles official installation on Ubuntu/Debian and Alpine.
- **Routing**: Configures IPv4/IPv6 forwarding and advertises specified subnet routes.
- **Auto-Update**: Enables Tailscale's built-in auto-update feature.
- **Status**: Reports login URLs or node status to the Ansible output.
